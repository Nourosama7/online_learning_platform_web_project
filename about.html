intro to CSE211 course

Welcome to the CSE211 Web Programming course.This course introduces students to the principles and practices of modern web application development,The curriculum covers the complete lifecycle of building web applications,students will have built and deployed a full-stack web application to the cloud, gaining both practical skills and real-world project experience.

Course Objectives

  1. Understand how to design and develop responsive, accessible, and semantic websites using HTML5 and CSS3, including Flexbox and Grid layouts
  2. implement interactivity using modern JavaScript,including DOM manipulation,events, form validation and API integration
  3. Build dynamic front-end applications using React with components, state management and client-side routing
  4. Develop server-side applications and RESTful APIs using Node.js and the Express framework
  5. Integrate relational databases (SQLite/Postgres) and perform CRUD operations
  6. Use modern development tools and workflows, including Git/GitHub, VS Code, and cloud deployment pipelines.

Topics Covered

  • Web development life cycly
  • Lists , Links , multimedia
  • Tables and Forms
  • identification , classes
  • Semantic Layout
