Course Overview

This intensive course prepares you to become a web developer and build professional web applications. You will work with HTML, CSS, JavaScript,databases

What You'll Learn

  1. Build websites using HTML & CSS
  2. JavaScript programming for manipulation
  3. Backend development using Node.js & Express
  4. Database systems

Requirements

  • no coding experience needed
  • A computer with internet connection
  • Basic computer usage skills

course duration: 10 Weeks (4 hours per week)

Course Syllabus

Module 1: Frontend Basics

Lesson 1: Introduction to HTML

Learn about tags, structure, and semantic elements

Lesson 2: CSS Layouts

Work with selectors, properties, flexbox, and grids.

Lesson 3: JavaScript Essentials

Variables, functions, loops, and event handling.

Module 2: Backend Development

Lesson 1: Node.js Overview

Understanding servers, NPM, and backend logic.

Lesson 2: Express.js Routing

Build routes, middleware, and request handlers.

Lesson 3: Working with MongoDB

CRUD operations, schema design, and database integration.
